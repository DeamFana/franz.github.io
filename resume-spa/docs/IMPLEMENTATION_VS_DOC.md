> 生成时间: 2026-03-18

## 一、架构变更（重大）

### 1.1 静态导出模式 → 混合模式

| 对比项 | 文档描述 | 实际实现 |
|--------|----------|----------|
| 部署模式 | 纯静态导出，`output: 'export'` | 开发模式用 API，生产用静态导出 |
| 数据获取 | 仅嵌入 HTML | 嵌入数据 + API 双通道 |
| 配置文件 | `next.config.ts` (带 output: export) | 拆分为两个：`next.config.ts`(开发) + `next.config.export.ts`(静态) |

**详细说明**:
- 文档描述使用 `output: 'export'` 进行纯静态导出
- 实际在 3月18日进行了重大变更：恢复了后端运行
- 新增 API 路由：`/api/resume` (GET/POST) 和 `/api/generate`
- 数据获取逻辑：前端优先读取 `window.__RESUME_DATA__`（静态HTML预填充），fallback 到 API
- 静态导出仍可用（通过 `npm run build:static`），但不再是唯一模式

/edit             → 编辑模式
