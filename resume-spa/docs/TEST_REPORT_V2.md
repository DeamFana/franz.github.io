## 改进计划

### 优先级 P0 (必须修复)

#### 1. 优化"生成静态页面"按钮用户体验

**问题**: 点击后无反馈，用户不知道是否成功

**改进方案**:
```typescript
// 添加状态反馈
const [generating, setGenerating] = useState(false);

// 点击处理
const handleGenerate = async () => {
  setGenerating(true);
  try {
    const res = await fetch('/api/generate', {
      method: 'POST',
      body: JSON.stringify({ jsonData: currentData })
    });
    const result = await res.json();
    
    if (result.success) {
      alert('✅ 静态页面已生成并部署！');
    } else {
      alert('❌ 生成失败: ' + result.error);
    }
  } catch (e) {
    alert('❌ 请求失败');
  } finally {
    setGenerating(false);
  }
};
```
